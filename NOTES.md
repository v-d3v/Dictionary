# Заметки по проекту

## API

### Free Dictionary API
Адрес: https://api.dictionaryapi.dev/api/v2/entries/en/{слово}
Ключ не нужен, бесплатно
Возвращает: определения, транскрипцию, часть речи, примеры, ссылку на аудио

### Random Word API
Адрес: https://random-word-api.herokuapp.com/word
Ключ не нужен, бесплатно
Возвращает: случайное английское слово

## Функции приложения
- Поиск определений слова
- Транскрипция и произношение
- Воспроизведение аудио прямо в приложении
- Кнопка случайного слова
- История последних 5 запросов

## Стек технологий
- C# .NET Framework 4.8
- WPF
- Free Dictionary API
- Random Word API
- NAudio
