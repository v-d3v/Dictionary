# Dictionary

WPF-приложение на C# (.NET Framework 4.8) для поиска определений английских слов.

## Что умеет

- Искать определения английских слов
- Показывать транскрипцию и часть речи
- Приводить примеры использования
- Воспроизводить произношение (кнопка 🔊)
- Хранить историю последних 5 запросов

## Технологии

- C# / WPF / .NET Framework 4.8
- [Free Dictionary API](https://dictionaryapi.dev/) — бесплатно, без ключа

## Как запустить

1. Открыть `Dictionary.sln` в Visual Studio
2. Нажать F5
